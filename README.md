Advanced OSINT (Open Source Intelligence) tool for searching usernames across multiple platforms.

## Features ✨

- Search username on **25+ platforms**
- Instagram, Facebook, Twitter/X, TikTok, GitHub, Reddit, and more
- Save results to file
- Search multiple usernames at once
- Cross-platform (Windows, Linux, Termux)
